## 2. `CpTdUtil` — 업무키 등록

사용법 요약(원본 시트 안내문):
- 각 시트 = 하나의 TR 오브젝트
- `Start` 시트에서 `TradeInit` 실행 후 계좌 비밀번호 입력
- 각 입력필드 채우고 "데이터 요청" 버튼 클릭 시 해당 오브젝트 실행
- `Event` 시트는 체결/정정/취소 발생 시 이벤트 수신 예제
